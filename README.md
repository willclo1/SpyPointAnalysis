# SpyPointAnalysis


## About
This is the backend to analyze spypoint game camera images. It identifies animals using the SpeciesNet AI Model(linked at the bottom). 
It will do basic cleaning work on the data and place the data into a csv and then put the data into a google drive. 

## Stack

DB: Google Drive(I know not great)

Tools: Google Vision OCR, Python, SpeciesNet
